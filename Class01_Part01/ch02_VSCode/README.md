# Ch02-07. 단축키 & 공백2로 만들기
- CMD + B: 사이드바 
- CMD + P: 빠른 열기
- CMD + Shift + P: 모든 명령
- CMD + W: 현재 탭 닫기
- CMD + F: 찾기
- CMD + OPT + F: 바꾸기
- OPT + 위아래방향: 줄이동
- OPT + Shift + Down/Up: 줄복사
- TAB: Indent(권장 size:2)
- CMD + Shift + [ or ]: 탭이동
- `CMD + \`: 창분할
> cf, CMD > Ctrl, OPT > Alt
